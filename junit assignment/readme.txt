JUNIT assignment

SRC/main folder:

The folder contains the three classes from the org.apache.commons.lang3.math library.

SRC/test folder:

The folder contains three test classes respectively for the three classes in the main folder. 

result folder:

The result folder contains the jacoco report generated by running the maven project.

pom.xml:

The pom.xml file includes dependencies for junit4 and also includes the surefire plugin and the jacoco plugin.
